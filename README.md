This repository contains a real-time face detection desktop application developed in Microsoft Visual Studio 2022 using C# Windows Forms and the OpenCvSharp library.

The application processes live webcam video and detects faces using OpenCVâ€™s pre-trained Haar Cascade classifier.

Detected faces are marked with rectangles on a PictureBox component in real time.
